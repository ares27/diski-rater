# DiskiRater

DiskiRater is a Progressive Web App (PWA) built with React, TypeScript, and Vite, designed for soccer/footy players to rate others. Users can rate each other based on criteria such as technical characteristics, skills, physicality, and overall performance. The app features offline support, automatic updates, and a responsive design powered by Bootstrap.

## Features

- Rate and review fellow players
- Progressive Web App with service worker for offline functionality
- Responsive UI using Bootstrap
- Built with modern tools: React, TypeScript, Vite
